# IAM Engineering Portfolio — NexusBridge Consulting

Portfolio project demonstrating real-world IAM engineering scenarios for a fictional cloud consulting company.

## Structure

```
00-company-context/   — Fictional company profile, org chart, roster, AWS setup
01-user-lifecycle/    — Joiner / mover / leaver automation
02-access-certification/ — Recertification campaigns
03-rbac-design/       — Role-based access control design
04-federation-sso/    — SAML/OIDC federation & SSO
05-secrets-rotation/  — Automated secrets rotation
06-incident-response/ — IAM incident response playbooks
```
