# AWS Certified Security – Specialty (SCS-C02)

The closest AWS certification focused on IAM. There is no standalone "AWS IAM Engineer" cert — this is the one recruiters look for IAM depth on AWS.

## IAM Topics Covered

- IAM policies, roles, and permission boundaries
- Federation (SAML, OIDC, AD, IAM Identity Center)
- Directory services (AWS Managed AD, AD Connector)
- Cross-account access strategies
- Resource-based vs identity-based policies
- STS, KMS integration with IAM
- Access analytics (IAM Access Analyzer, CloudTrail)

## Additional Exam Domains

- Incident response (forensics, compromised resources)
- Logging and monitoring (CloudWatch, CloudTrail, GuardDuty)
- Infrastructure security (VPC, security groups, NACLs, WAF)
- Data protection (S3 policies, KMS, CloudHSH, Macie)

## Details

- **Code:** SCS-C02
- **Cost:** $300 USD (often 50% discount with free practice exam via AWS)
- **Recommended experience:** 2+ years hands-on AWS security experience
- **Format:** 65 questions, 170 minutes, multiple choice/multiple response
- **Validity:** 3 years
