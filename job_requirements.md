# IAM Engineer — Common Job Requirements

## Core Technical Skills

| Category | Technologies |
|----------|-------------|
| **Identity Platforms** | Microsoft Entra ID (Azure AD), Okta, SailPoint IdentityIQ/Identity Security Cloud, Saviynt |
| **PAM** | CyberArk, BeyondTrust PRA/BeyondTrust PAM |
| **Protocols** | OAuth 2.0, OpenID Connect, SAML, SCIM |
| **Directory Services** | Active Directory, Entra ID, LDAP |
| **Scripting/Automation** | PowerShell, Python, Terraform, Okta Workflows/Expression Language |
| **Cloud** | AWS IAM, Azure Policy, Azure landing zones |
| **Secrets Management** | HashiCorp Vault, Azure Key Vault |

## Common Duties

- Design and implement IAM platforms (identity lifecycle, provisioning, de-provisioning)
- RBAC, Segregation of Duties (SoD), Access Recertification
- User-to-service & service-to-service authentication
- Federation services integration (SSO, MFA, conditional access)
- Identity governance & compliance (audit walkthroughs, access reviews)
- Secrets management — secure storage, rotation, auditability
- 3rd-line support, incident response, on-call rotation
- IAM strategy, roadmap, and reference architecture
- Stakeholder collaboration (HR, IT, Security, DevOps, Legal)

## Certifications Commonly Requested

- Microsoft Certified: Identity and Access Administrator Associate (SC-300)
- CISSP
- Microsoft Cybersecurity Architect Expert
- CIAM or equivalent identity certifications
- ITIL Foundation

## Experience Levels

| Level | Years | Typical Salary (London) |
|-------|-------|------------------------|
| **Junior/Support** | 1–3 yrs | £50k–£64k |
| **Engineer** | 3–5 yrs | £65k–£85k |
| **Senior Engineer** | 5–8 yrs | £85k–£120k |
| **Lead/Architect** | 8+ yrs | £100k–£125k+ or £500–£750/day contract |

## Soft Skills

- Stakeholder management / translating security risks to non-technical audiences
- Mentoring and leading junior engineers
- Agile delivery (sprint planning, backlog management)
- Documentation and process definition
- Proactive problem-solving, minimal supervision
